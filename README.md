# 学堂在线助手

> 学堂在线课程自动化脚本，支持刷课、答题、题库收集与导入导出。

## 版本

**2.0.0**

## 作者

yangnianboy

## 功能

| 功能 | 说明 |
|------|------|
| **刷课** | 自动播放视频（静音）、翻附件、跳过测验，视频播放到 95% 自动跳转下一节 |
| **收集** | 自动遍历所有题目，收集答案并存入本地题库（选择题/判断题/填空题） |
| **答题** | 根据题库自动匹配并作答，支持单选、多选、判断、填空 |
| **破解(小号)** | 随机选择答案快速提交，**不收集答案**，适合小号刷出题库为**收集**做准备 |
| **题库** | 点击展开/收起题库面板，查看已收集的所有题目和答案 |
| **清空** | 一键清空本地题库 |
| **导出** | 将题库导出为 JSON 文件，方便备份和分享 |
| **导入** | 从 JSON 文件导入题库，自动去重合并 |
| **拖拽** | 拖动标题栏自由调整面板位置 |
| **折叠** | 点击标题栏 `−` 按钮收起/展开面板 |

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 打开 [学堂在线助手-2.0.0.user.js](学堂在线助手-2.0.0.user.js)
3. 点击 Tampermonkey 弹出窗口中的「添加新脚本」

## 使用

1. 打开 [学堂在线](https://www.xuetangx.com/) 课程页面
2. 页面右侧会出现控制面板
3. 点击对应按钮启动功能，再次点击停止

### 按钮说明

```
┌──────────────────────┐
│   学堂在线助手   [−]  │  ← 拖拽移动 / 折叠面板
├──────────────────────┤
│   刷课   │   收集    │
│ 破解(小号) │   答题    │
│   题库   │   清空    │
│   导出   │   导入    │
├──────────────────────┤
│   题目    │   答案    │  ← 题库表格（点击「题库」展开）
├──────────────────────┤
│   xxx    │    C     │
└──────────────────────┘
```

### 运行模式

- 点击功能按钮进入运行模式，按钮变为绿色并显示「停止」
- 同一时间只能运行一个模式，其他按钮自动禁用
- 再次点击同一按钮或点击其他模式按钮即可停止当前模式
- **收集**模式下题库表格自动展开
- **破解(小号)**模式下题库表格自动收起，且不会收集答案

## 题库格式

```json
[
    {
        "type": 1,
        "timu": "题目内容",
        "anwers": ["C"]
    }
]
```

| type | 题型 |
|------|------|
| 1 | 选择题（答案为选项字母数组） |
| 2 | 判断题（0 = 正确，1 = 错误） |
| 3 | 填空题（答案为文本数组） |

项目中附带了一份预收集的题库 [xuetangx-anwers.json](xuetangx-anwers.json)，可通过「导入」按钮加载使用。
